
### 汉明距离

#### 两个整数之间的 汉明距离 指的是这两个数字对应二进制位不同的位置的数目。

```
示例1
输入：x = 1, y = 4  
  输出：2  
  解释：  
  1   (0 0 0 1)  
  4   (0 1 0 0)  
         ↑   ↑  
  上面的箭头指出了对应二进制位不同的位置。

示例2
输入：x = 3, y = 1
输出：1
```

---

思路一: 
1.异或运算, 然后用公式n & (n-1)
---

```
var hammingDistance = function(x, y) {
    let s = x ^ y, ret = 0;
    while (s != 0) {
        s &= s - 1;
        ret++;
    }
    return ret;
};
```
